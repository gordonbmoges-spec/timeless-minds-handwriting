# Magical archive material sources

The files in this folder were generated for this project with OpenAI image generation on 2026-07-19.

- `enchanted-archive.png`: photoreal empty archive and bookshelf background plate.
- `archive-with-mirror.webp`: brighter archive plate with a shelf-mounted enchanted mirror, generated by editing the original archive plate.
- `books/*.webp`: eight independently generated, photoreal 3D magical-book props for the six historical figures, Tom Riddle diary, and human parchment. Generated with a shared candlelight/material brief, then mechanically isolated to transparent WebP assets; labels and interactions remain live HTML.
- `covers/*.webp`: eight independently generated, straight-on photoreal magical-book covers for the shelf display and opening animation. They share one orthographic camera and warm museum-lighting brief; readable labels remain live HTML.
- `panoramic-parchment.webp`: full-bleed parchment texture with no visible sheet boundary, generated as the immersive handwriting surface.
- `antique-leather-spine.png`: photoreal blank antique leather spine material.
- `blank-parchment.png`: photoreal blank vellum writing surface.

The prompts deliberately exclude existing franchise marks, characters, readable text, and logos. These files are used as visual material layers; all book labels, controls, writing, and page interaction remain live HTML/canvas content.
